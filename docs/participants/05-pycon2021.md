4. Select your pronouns in `#pronouns` and choose roles in `#roles`
5. **Make sure to  use your real name or your GitHub handle**

## Getting ready for the sprint

This might be your first sprint so make sure to check the other sections in this handbook.
Particularly make sure to install all needed tools as described in [What do I need?](http://127.0.0.1:8000/participants/01-index/#what-do-i-need-to-prepare-before-joining-the-sprint)

In general you will need:

1. A computer with Internet connection
2. Git installed
3. A text editor / IDE
4. A working development environment for the project you will be working on

We have listed the participating projects in this page. We highly recommend you to come prepared
and check the **Contributing guidelines** as well as the **Setting up your development environment** before the sprint.
